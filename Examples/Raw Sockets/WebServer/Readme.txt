A simple webserver made in Sputnik it can display pages, images let people download files etc.

You can also place Sputnik code in <?spk echo "code here"; ?> similar to PHP

It also has a template folder for showing HTTP error codes to a client

It also processeds the ?key&value query into an array and prints it

Open the WebServer and connect to it using http://127.0.0.1:8080
